# Sistema de Gerenciamento de Tarefas

Projeto desenvolvido em Python para a disciplina de Fundamentos e Técnicas Avançadas.

## Funcionalidades
- Adicionar tarefas
- Listar tarefas
- Marcar tarefas como concluídas
- Remover tarefas
- Menu interativo no terminal

## Tecnologias utilizadas
- Python 3

## Como executar
1. Clone o repositório
2. Execute o arquivo `main.py`
3. Interaja pelo terminal
